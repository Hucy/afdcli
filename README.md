## 概述
  
  修改[vue-cli](https://github.com/vuejs/vue-cli) 而来 用生成模板文件目录

## 用法 

  1. clone 仓库到本地：

    ```
      git clone https://github.com/Hucy/afdcli.git
    ```
  2.  进入 `afdcli` :

    ```
      cd afdcli
    ```
  3. 执行 ` npm i . -g`将 模块安装到全局环境

  4. 命令行执行 ` agi ` 查看帮助


## 自定义模板
  
  模板存放路径为 ` templates `, 可在 `templates` 文件夹下新建文件目录,`meta.json`文件是模板变量替换.
